## Hexo
### local run
 + npm install hexo-cli -g
 + hexo init blog
 + cd blog
 + npm install
 + hexo server
 
### write and deploy
 + hexo new "My New Post"
 + hexo clean
 + hexo generate
 + hexo deploy
